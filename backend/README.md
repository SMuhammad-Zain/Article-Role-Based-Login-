# Article Sharing App

---

## Framework\Libraries

    * Express 5.2.1: Backend Framework

    * Crypto 1.0.1: for hashing passwords and refresh token

    * Mongoose 9.7.1: ODM, Schemas and Models

    * JsonWebToken 9.0.3: For creating a timed tokens

---

### Feautures

    * Role Based Login

    * Authorization before creating and ammending posts

    * Access and Refresh tokens for proper logins

    * Session logging to allow multiple logins
